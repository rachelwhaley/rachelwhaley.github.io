# Waste Not
##  Schools As Agents of Pro-Environmental Behavior Change in Chicago Communities 

This project mapped publicly available data from Chicago's Blue Cart residential recycling program and recycling in a randomly-selected sample of Chicago's public schools in 2014, and was published in the [Chicago Studies Journal, Vol. 9](https://humanities-web.s3.us-east-2.amazonaws.com/college/chicagostudies-prod/s3fs-public/2020-05/14%20Chicago%20Studies%20V9.pdf).

All data used for this project was openly available at the following links:

- Chicago Public Schools recycling data by school: www.cpsrecycles.com.
- Chicago Blue Cart residential recycling data: [Chicago Department of Streets and Sanitation Blue Cart page](http://www.cityofchicago.org/city/en/depts/streets/supp_info/recycling1/about_blue_cart_recycling.html).
- Chicago Public Schools student demographics by school: [Chicago Public Schools data page](http://cps.edu/SchoolData/Pages/SchoolData.aspx).
